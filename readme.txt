WHAT THIS APP DOES

This app connects Homey to your CheckWatt EnergyInBalance account. You can see battery level, net income for the site, and site status on the device tile, in Insights and in Flows. Sign in with the same username and password you use for the EnergyInBalance service. This app is made by the community, is not official Checkwatt software, and uses unofficial access to the same web APIs the mobile and web clients use. Credit for API understanding belongs in the app manifest contributors section.


HOW TO INSTALL FROM THE HOMEY APP STORE

When the app is available in the Homey App Store, open Homey on your phone or in the browser, search for the app, tap Install, and follow the prompts. Then add a device under the app. Enter your EnergyInBalance username and password when asked—you can also change or re-enter them later under the device settings for this app. You only use Homey's own screens for this; you do not need developer tools or a manual install from source.


FOR DEVELOPERS AND TEST BUILDS

If you install from the project source yourself, you need Node.js and the Homey CLI on your computer. Full step-by-step commands, validation, and remote install are described in the README file in this project's GitHub repository (README.md).

VAD APPEN GÖR

Appen kopplar Homey till din CheckWatt-anläggning i EnergyInBalance. Du kan följa batterinivå, nettointäkter för anläggningen, och status på enhetens platta, i Insights och i flöden. Logga in med samma användarnamn och lösenord som till tjänsten EnergyInBalance. Appen är community-utvecklad, är inte officiell från CheckWatt och använder inofficiell åtkomst till samma webb-API:er som de officiella klienterna. Tack till bidragsgivare står i appmanifestet under contributors.


INSTALLATION FRÅN HOMEY APP STORE

När appen finns i Homey App Store: öppna Homey på mobilen eller i webbläsaren, sök efter appen, välj Installera och följ guiden. Lägg sedan till en enhet under appen. Ange ditt användarnamn och lösenord för EnergyInBalance när du uppmanas—du kan också ändra eller ange dem senare under enhetens inställningar för appen. Du använder bara Homeys egna skärmar; du behöver inga utvecklarverktyg eller manuell installation från källkod.


UTVECKLARE OCH TESTVERSIONER

Om du installerar från projektets källkod behöver du Node.js och Homey CLI på datorn. Fullständiga steg-för-steg-kommandon, validering och fjärrinstallation finns i README-filen i detta projekts GitHub-repo (README.md).
